<b>Connecting Vision and Language with Localized Narratives,</b><br>
Jordi Pont-Tuset, Jasper Uijlings, Soravit Changpinyo, Radu Soricut, and Vittorio Ferrari,<br>
arXiv 1912.03098, 2019.<br>
